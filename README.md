# 🏠 Arquitetura Homelab

> Repositório central de documentação da arquitetura do meu Homelab.

Este repositório reúne a visão geral da infraestrutura, diagramas, decisões de arquitetura e referências para os projetos que compõem o ambiente. Cada serviço ou solução possui (ou terá) seu próprio repositório, permitindo uma documentação mais organizada e independente.

## 🎯 Objetivo

Centralizar a documentação da arquitetura do Homelab, servindo como ponto de partida para consulta, estudo e evolução da infraestrutura.

## 📚 Conteúdo

Em breve este repositório conterá:

- Arquitetura geral do ambiente
- Topologia da rede
- Inventário de hardware
- Serviços hospedados
- Containers e máquinas virtuais
- Estratégias de backup
- Monitoramento e observabilidade
- Segurança
- Diagramas da infraestrutura
- Links para os repositórios relacionados
